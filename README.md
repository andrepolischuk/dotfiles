# dotfiles

> Dotfiles for shell, zsh, vim, macOS, etc

![](screenshot.png)

## Install

```sh
# dotfiles
git clone https://github.com/andrepolischuk/dotfiles && ~/dotfiles/init.sh

# git config
git config --global user.name "Your Name"
git config --global user.email "email@yoursite.com"
```

You may set macOS defaults:

```sh
~/dotfiles/macos.sh
```

## Apps

* [zsh][zsh]
* [oh-my-zsh][oh-my-zsh]
* [git][git]
* [vim][vim]
* [min zsh prompt][min]

## Update

You can run for update all apps:

```sh
~/dotfiles/update.sh
```

* dotfiles
* zsh
* zsh plugins
* brew packages
* vim plugins
* global npm packages
